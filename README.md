# Taller_3 Programa con menu infinito

## 📖 Descripción
Este proyecto consiste en desarrollar un programa que implemente un menú infinito, permitiendo al usuario seleccionar diferentes operaciones matemáticas.

El programa incluye las siguientes funcionalidades:

- Calcular el n-ésimo número de Fibonacci
- Encontrar el n-ésimo número primo
- Resolver las raíces de una ecuación cuadrática
- Calcular el promedio de N números

El sistema debe validar la entrada de datos y manejar posibles errores para evitar fallos durante la ejecución.
